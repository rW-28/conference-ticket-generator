# conference-ticket-generator
Simple landing page with HTML, CSS, and JS

A simple front-end project built with **HTML, CSS, and JavaScript**, where users can generate a personalized ticket for **Coding Conf 2025**.  
The app collects user data, including name, email, GitHub profile — then dynamically creates a custom conference ticket.

Features
- Interactive form with input fields for name, email, and GitHub.
- Data transfer between pages using `localStorage`.
- Automatically generated unique ticket number.
- Responsive layout and  design with gradients and patterns.

Technologies Used
- **HTML5** – structure and content  
- **CSS3** – styling and responsive layout  
- **JavaScript (ES6)** – dynamic data handling and localStorage API

  What I Learned
- Passing data between pages using **`localStorage`**.
- Designing and structuring a small front-end project for real-world scenarios.
- Publishing a static website with **GitHub Pages**.
